    「灰原？」我叫着一个熟悉的名字，不太清楚它的意义，但是我的嘴却就这么脱口而出了。朝周围望望，四周无人，只我一人站在空旷的黑寂之中。

    「工藤……」微微的声音在我的耳边回响着。这个声音是……灰原？我迅速地睁开眼睛才发现她正背对着我。

    笨蛋，你去哪了！我大声地向她吼道，而眼眸中那茶色的秀发却显得如此的陌生。「我要走了。」我的眼前闪过几个响亮的大字。

    我所谓的富贵家庭不是一般的有钱而已，爸爸三云慎吾坐拥整个日本的绝大多数股票份额，公司甚至于遍布整个全世界。而我整个的成长经历也几乎没有遇到过任何的困难，或者就算有也早就被宠爱我的爸爸给一扫而空了。

    现在的我13岁了，可是长这么大依旧是没有怎么接触过外界的人，而用爸爸的话说就是「你是富贵人家的千金，怎么能和那种外面的野孩子一起玩呢？」于是从小到大我都被爸爸严格地监视着，或者更准确地说……是被爸爸雇来的保镖监视着，他们护送我上下学，而除了在学校的时间，我几乎都被锁在家里。

    在学校里，我的成绩十分的优异，常年在学校里拿年级第一，而这也得益于爸爸在家中为我所请的老师来为我进行详细的功课辅导。另外，和学校里的那些所谓的「美丽的女孩」相比，我自认我的相貌更是比她们高出一筹。别看我年纪尚青，但是身材的发育乃至于整个人线条的比例也是同龄人所难以比拟的。而这些……或许就是我那个从未见过的妈妈所遗传而来的吧。

    可能就是因为我出众的学习成绩和完美的身材，从小学开始到现在的初中，我的追求者可以说是数不胜数，鞋柜几乎天天也都是被各式各样的情书所充满的，不过我连看都不看就会让我的保镖将它们处理掉。
