    “呵呵，是吗……”女孩露出一种诡异的表情，并用半圆形的眼睛看着他。“我可不想有人说我谋杀侦探啊，要是我们再呆在这里，我可害怕大侦探的鼻血可是要流光了……”说罢，女孩用手调皮地按了按男孩鼻中还塞着的纸团。
